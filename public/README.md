# Thư mục Public

Đặt ảnh Bác Hồ vào đây với tên file: `ho-chi-minh.png`

Ảnh sẽ được hiển thị thay cho emoji trong ứng dụng.

**Yêu cầu:**
- Tên file: `ho-chi-minh.png`
- Format: PNG (hoặc JPG/JPEG)
- Kích thước khuyến nghị: 200x200px hoặc lớn hơn (ảnh sẽ được resize tự động)
- Ảnh nên có nền trong suốt hoặc nền trắng để hiển thị đẹp trong avatar tròn
